# Experiments Directory (Plan #277)

This directory contains experiment configurations for testing different agent motivation configurations and observing emergent behavior.

## Files

| File | Purpose |
|------|---------|
| `README.md` | Experiment workflow documentation |
| `TEMPLATE.yaml` | Template for new experiments |
| `exp_001_discourse_baseline.yaml` | First experiment: discourse analyst baseline |

## Creating Experiments

1. Copy `TEMPLATE.yaml` to a new file (e.g., `exp_002_name.yaml`)
2. Configure agents and motivation profiles
3. Run the experiment
4. Record observations in the experiment file
