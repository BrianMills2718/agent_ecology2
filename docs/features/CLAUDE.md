# Feature Documentation Directory

Documentation for specific features of the Agent Ecology system.

## Purpose

Feature docs explain how specific features work for users and developers. These supplement the architecture docs with practical usage details.

## File Structure

| File | Content |
|------|---------|
| `dashboard.md` | Real-time HTML dashboard with WebSocket updates |
| `mint_auction.md` | Vickrey auction system for artifact scoring |

## Writing Feature Docs

Feature docs should include:
1. **What it does** - Brief description
2. **How to use** - Practical examples
3. **Configuration** - Relevant config options
4. **Architecture** - How it fits into the system

## Related Directories

- `features/*.yaml` - Feature definitions (code mappings)
- `docs/architecture/current/` - How features are implemented
- `docs/architecture/target/` - Where features are going
